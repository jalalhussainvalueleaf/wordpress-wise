Easily add, manage, and display FAQs for individual WordPress posts with a modern admin UI and REST API support.

== Description ==
Post FAQ Manager is a user-friendly plugin that allows you to easily add, manage, and display Frequently Asked Questions (FAQs) for individual WordPress posts. With a modern and intuitive interface, you can enable or disable FAQs per post, add unlimited questions and answers, and manage them directly from the post editor. The plugin stores FAQ data as post meta and exposes it via the WordPress REST API for easy integration with themes and custom front-end displays.

= Features =
* Enable or disable FAQs for each post with a simple checkbox.
* Add, edit, and remove unlimited FAQ items per post.
* Clean, professional, and responsive admin UI that blends with the WordPress dashboard.
* All FAQ data is saved as post meta for easy retrieval and theme integration.
* FAQ data is available in the WordPress REST API for headless or custom front-end use.
* No coding requiredâ€”just install and start managing FAQs!

= Perfect for =
Bloggers, businesses, and developers who want to add structured FAQs to their posts without complex setup or coding.

== Frequently Asked Questions ==
= How do I enable FAQs for a post? =
Edit any post and check the "Enable FAQ Options" box in the FAQ Manager meta box.

= Where is the FAQ data stored? =
All FAQ data is stored as post meta and is available via the REST API.

= Can I use this with custom post types? =
Currently, it works with standard posts. For custom post type support, contact the author or submit a feature request.
