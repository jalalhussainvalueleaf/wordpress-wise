Manage and search Indian post office/pincode data easily from your WordPress admin. Supports CSV import, filters, editing, and download.

== Description ==

**Pincode Dashboard** provides a powerful and user-friendly admin interface to manage India’s post office/pincode data directly from the WordPress backend. Easily upload large CSV files, search and filter by state/district/post office, and download complete data.

**Features:**

- Bulk CSV upload with progress bar
- Filter by State, District, and Post Office
- Editable rows and inline saving
- Delete individual or all entries
- Download full dataset in CSV format
- Pagination and responsive design
- No duplicate restriction on pincode (allows overlapping values)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pincode-dashboard/` directory or install via WordPress admin.
2. Activate the plugin through the ‘Plugins’ screen.
3. Navigate to **Tools > Pincode Dashboard** to start managing data.
