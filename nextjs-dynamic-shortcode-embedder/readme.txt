Dynamically register shortcodes for Next.js components and manage them from the admin.

== Description ==
This plugin allows you to dynamically register and manage WordPress shortcodes that embed Next.js components. Easily add, edit, or remove shortcodes from the admin settings page, and use them in your posts or pages to render dynamic content from your Next.js app.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/nextjs-dynamic-shortcode-embedder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Next.js Shortcode Embedder to configure your shortcodes and Next.js base URL.

== Frequently Asked Questions ==
= What does this plugin do? =
It lets you create and manage shortcodes that embed Next.js components in your WordPress content.

= Can I use this with any Next.js app? =
Yes, as long as your Next.js app supports embedding components via a URL.
