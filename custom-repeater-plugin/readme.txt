Adds a repeater field functionality to the WordPress editor.

== Description ==
This plugin adds a repeater field to the WordPress editor, allowing you to create and manage dynamic sets of fields in your posts or pages. Useful for custom content structures and flexible data entry.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-repeater-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure and use the repeater fields in the post editor.

== Frequently Asked Questions ==
= What is a repeater field? =
A repeater field allows you to add multiple sets of fields dynamically in the editor.

= Can I use this with custom post types? =
Yes, repeater fields can be used with any post type that supports the editor.
