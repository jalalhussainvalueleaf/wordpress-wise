Search and display IFSC code, branch details, and bank information using shortcode or dashboard.

== Description ==

The IFSC Dashboard plugin allows you to manage and display Indian bank details including IFSC, MICR, branch address, contact info, and more.

With a user-friendly dashboard interface and support for CSV uploads and downloads, itâ€™s perfect for administrators managing large bank datasets.

== Features ==

* Search IFSC code and get complete bank details.
* Filter by Bank Name, State, District, Branch, or IFSC directly.
* Upload bank details using a CSV file.
* Download all data as CSV.
* Edit or delete rows from the dashboard with ease.
* Pagination support for large datasets.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ifsc-dashboard/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the admin sidebar and click on **IFSC Dashboard** to begin managing your data.

== Frequently Asked Questions ==

= Can I search by IFSC code directly? =
Yes, there's a dedicated input for searching by IFSC code in the dashboard.

= What format should my CSV be in? =
The CSV must contain columns like Bank Name, Branch, IFSC, MICR, Contact Number, Address, and City/State.

= Will it overwrite duplicate IFSC codes on CSV upload? =
You will get a prompt asking whether to update existing entries or skip duplicates.
